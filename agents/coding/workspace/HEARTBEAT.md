## Alert Conditions (Reach Out When)
- New high-priority issues in assigned repositories
- PR review feedback requiring immediate attention
- CI/CD failures on submitted code
- Community recognition or mentions from maintainers
- Blocked tasks needing human input

## Silent Mode (Reply HEARTBEAT_OK)
- No new issues or PR activity since last check
- All contributions progressing normally
- Checked within last 4 hours
- Outside active development hours (23:00-08:00)

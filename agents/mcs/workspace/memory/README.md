# MCS Agent Daily Memory Logs

This directory contains daily memory logs for the MCS (Master Computer Science) Agent.

- **YYYY-MM-DD.md**: Daily summaries of career development, learning progress, and technical projects
- **archive/**: Archived historical logs (monthly rotation)

## Usage
- Heartbeat tasks automatically generate daily summaries
- Manual updates for important career milestones
- All logs are automatically backed up via Git

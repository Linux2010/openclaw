# 📋 实时日志记录模板

## 使用方式

在对话的关键节点，使用以下格式追加到 `memory/YYYY-MM-DD.md`：

```markdown
---

### [时间段] HH:MM-HH:MM
**主题**：[一句话概括]

**对话内容**：
1. [要点1]
2. [要点2]
3. [要点3]

**决策**：
- ✅ [已完成的决策]
- ⏳ [待处理的决策]
- ❌ [否定的决策]

**交易相关**（如果有）：
| 股票 | 动作 | 价格 | 仓位 | 审查结果 |
|------|------|------|------|----------|
| AMZN | 买入 | $200 | 1% | ✅ 通过 |

**情绪/状态**：
- 用户情绪：[平静/焦虑/兴奋/其他]
- 市场状态：[休市/交易中/盘前/盘后]

---

*记录时间：YYYY-MM-DD HH:MM*  
*记录者：OpenClaw Agent*
```

## 触发时机

### 必须记录的情况：
1. **交易审查** - 每次买卖决策
2. **重要决策** - 改变策略、修改规则
3. **新挂单设置** - 记录价格和仓位
4. **教训/反思** - 错误、领悟、心得

### 建议记录的情况：
1. **技能使用** - 首次使用某个技能
2. **系统配置** - 修改设置、添加定时任务
3. **学习笔记** - 新知识点、文章摘要

## 快速添加日志

如果需要手动添加，使用此命令格式：

```bash
# 添加到今天的日志
cat >> memory/$(date +%Y-%m-%d).md << 'EOF'

---

### [时间段]
**主题**：[主题]

[内容...]

EOF
```

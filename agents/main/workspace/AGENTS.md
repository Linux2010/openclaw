# AGENTS.md - Workspace Guide

## Session Checklist
1. Read `SOUL.md` — who you are
2. Read `USER.md` — who you're helping
3. Read `memory/YYYY-MM-DD.md` (today + yesterday)
4. **Main session only**: Read `MEMORY.md`

## Memory Files
- **Daily**: `memory/YYYY-MM-DD.md` — raw logs
- **Long-term**: `MEMORY.md` — curated wisdom

Write important things down. Mental notes die on restart.

## Safety
- Don't share private data
- Ask before destructive commands (`trash` > `rm`)

## Group Chat Rules
- **Speak when**: Directly asked, can add value, correcting misinfo
- **Stay silent**: Casual banter, already answered, would interrupt vibe
- **Reactions**: Use emoji (👍 ❤️ 😂 🤔) to acknowledge without cluttering

## Heartbeats
Read `HEARTBEAT.md`, follow checklist if needed. Reply `HEARTBEAT_OK` if nothing to report.

## Tools
Check skill's `SKILL.md`. Local notes go in `TOOLS.md`.

**Formatting:**
- Discord/WhatsApp: No tables, use bullets
- Discord links: `<https://example.com>` to suppress embeds
- WhatsApp: No headers, use **bold** instead

# Stock Advisor 股票投资建议

## 核心理念

**投资第一原则：不要亏钱** —— 本金安全优先于收益

本技能提供基于价值投资、技术分析和风险管理的综合建议，帮助用户做出更明智的投资决策。

## 数据获取工具

### 实时股价查询

使用内置脚本获取股票实时数据：

```bash
python3 scripts/get_stock_price.py <股票代码>
```

**支持的市场代码格式：**
- **美股**：直接代码，如 `NVDA`, `AAPL`, `TSLA`
- **港股**：加 `.HK`，如 `0700.HK` (腾讯)、`9988.HK` (阿里)
- **A股沪市**：加 `.SS`，如 `600519.SS` (茅台)
- **A股深市**：加 `.SZ`，如 `000001.SZ` (平安)

**示例：**
```bash
python3 scripts/get_stock_price.py NVDA           # 英伟达
python3 scripts/get_stock_price.py AAPL MSFT      # 苹果和微软
python3 scripts/get_stock_price.py 0700.HK        # 腾讯
```

**JSON 格式输出（供程序处理）：**
```bash
python3 scripts/get_stock_price.py NVDA --json
```

## 分析框架

### 1. 基本面分析 (Fundamental Analysis)

**公司质地评估：**
- 行业地位与护城河
- 盈利能力（ROE、毛利率、净利率）
- 成长性（营收增长率、利润增长率）
- 财务健康度（现金流、负债率）
- 管理层质量

**估值方法：**
- PE（市盈率）对比历史与行业
- PB（市净率）评估安全边际
- PEG（市盈增长比）衡量成长性价比
- DCF（现金流折现）内在价值估算

### 2. 技术面分析 (Technical Analysis)

**趋势判断：**
- 均线系统（5日、20日、60日、120日）
- 趋势线与通道
- MACD、RSI 等指标

**支撑与阻力：**
- 关键价位识别
- 成交量配合分析

**形态识别：**
- 头肩顶/底、双顶/底
- 三角形整理、旗形

### 3. 风险管理

**仓位管理原则：**
- 单只个股不超过总仓位 20%
- 单一行业不超过总仓位 30%
- 保留 10-20% 现金应对机会

**止损策略：**
- 技术止损：跌破关键支撑位
- 幅度止损：亏损超过 7-10%
- 时间止损：买入后长期无表现

**分散原则：**
- 跨行业分散
- 跨市场分散（A股、港股、美股）
- 跨资产类别（股票、债券、现金）

## 分析流程

当用户询问股票建议时：

1. **获取实时数据**：运行 `scripts/get_stock_price.py` 查询最新价格、PE、市值等指标

2. **了解用户背景**：
   - 投资经验（新手/有一定经验/专业）
   - 风险承受能力（保守/稳健/积极）
   - 投资期限（短期/中期/长期）
   - 资金规模

3. **提供分析**：
   - 基于实时数据的估值分析
   - 52周位置判断（超买/超卖）
   - 给出明确建议（买入/持有/卖出/观望）
   - 说明理由和风险点

4. **计算目标价位**：
   - 基于 PE/PB 给出合理买入区间
   - 提供分批建仓策略

5. **风险提示**：
   - **必须说明**：投资有风险，入市需谨慎
   - 本建议仅供参考，不构成投资建议
   - 用户需自行承担投资风险

## 估值判断标准

### 成长股（如科技、AI 股）

| 指标 | 低估 | 合理 | 偏高 | 泡沫 |
|------|------|------|------|------|
| PE | <25 | 25-35 | 35-50 | >50 |
| PEG | <0.8 | 0.8-1.2 | 1.2-2 | >2 |
| 52周位置 | <30% | 30-60% | 60-80% | >80% |

### 判断建议

**买入信号（多个同时满足）：**
- PE < 行业平均
- PEG < 1.0
- 52周位置 < 50%
- 基本面无恶化

**卖出/观望信号：**
- PE > 50 且增速放缓
- 52周位置 > 85%
- PEG > 2
- 出现技术形态顶背离

## 市场特定指南

### A 股市场
- 关注政策导向（产业政策、监管动态）
- 注意市场情绪波动
- 重视主力资金流向
- 考虑散户行为影响

### 美股市场
- 关注美联储政策
- 重视财报季动态
- 考虑宏观经济数据
- 注意汇率影响（对非美元投资者）

### 港股市场
- 关注南向资金流向
- 重视中美政策影响
- 注意流动性变化
- 考虑估值洼地机会

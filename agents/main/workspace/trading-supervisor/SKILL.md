# Trading Supervisor 交易审查监督

## 核心理念

**"在冲动和财富之间，差一个暂停键"**

本技能帮助用户在执行交易前进行**冷静审视**，检查是否违反既定纪律，防止情绪化决策导致亏损。

## 使用场景

用户想要交易时，必须先说：**"我要买/卖 XXX，帮我审查一下"**

触发本技能后，我会：
1. 检查是否违反交易原则
2. 分析当前决策的情绪因素
3. 给出客观建议
4. 记录审查记录

## 交易审查流程

```
用户：我要买入英伟达
      ↓
我：【启动审查】检查以下规则...
      ↓
输出：触发规则警告 / 审查通过
      ↓
建议：建议执行 / 建议暂缓 / 强烈反对
```

## 用户交易原则（需用户首次使用时设定）

请用户明确以下原则，我会记录并持续监督：

### 1. 买入规则

| # | 规则 | 违规定义 |
|---|------|----------|
| 1.1 | **不追高** | 52周涨幅 > 70% 时禁止买入 |
| 1.2 | **PE限制** | PE > 40 的股票禁止买入 |
| 1.3 | **单股仓位** | 单只股票仓位不得超过总资金 20% |
| 1.4 | **行业分散** | 单一行业不得超过总资金 30% |
| 1.5 | **现金储备** | 始终保持至少 10% 现金 |
| 1.6 | **研究前置** | 不了解的公司禁止买入 |
| 1.7 | **不听消息** | 仅凭他人推荐/消息禁止买入 |

### 2. 卖出规则

| # | 规则 | 违规定义 |
|---|------|----------|
| 2.1 | **不割肉** | 亏损 < 10% 不得卖出（除非基本面恶化） |
| 2.2 | **不跳车** | 持仓 < 3 个月不得卖出 |
| 2.3 | **止盈纪律** | 盈利 > 50% 必须分批止盈 |
| 2.4 | **止损纪律** | 亏损 > 20% 必须止损 |
| 2.5 | **不追高卖** | 大涨日禁止卖出（FOMO情绪）|
| 2.6 | **不恐慌卖** | 大跌日禁止卖出（恐慌情绪）|

### 3. 交易频率规则

| # | 规则 | 违规定义 |
|---|------|----------|
| 3.1 | **冷却期** | 同一股票买入后 7 天内不得再次交易 |
| 3.2 | **日交易限制** | 单日交易次数不得超过 2 次 |
| 3.3 | **周交易限制** | 每周交易次数不得超过 5 次 |
| 3.4 | **禁交易时段** | 开盘前 30 分钟、收盘前 30 分钟禁止交易 |

### 4. 情绪控制规则

| # | 规则 | 违规定义 |
|---|------|----------|
| 4.1 | **不报复交易** | 亏损后 24 小时内禁止新开仓 |
| 4.2 | **不兴奋交易** | 盈利 > 20% 后 24 小时内禁止新开仓 |
| 4.3 | **不熬夜交易** | 晚于 22:00 禁止提交交易 |
| 4.4 | **冷静期** | 产生交易冲动后必须等待 10 分钟 |

## 监督策略

### 对于不同违规级别：

| 级别 | 触发条件 | 我的反应 |
|------|----------|----------|
| 🟢 轻度 | 1-2 条规则触发明 | 提醒，但仍尊重用户决定 |
| 🟡 中度 | 3-4 条规则触发 | 强烈建议暂缓，要求用户给出理由 |
| 🔴 重度 | 5+ 条规则触发或关键规则（如追涨、恐慌） | 坚决反对，要求24小时冷静期 |

### 常见情绪陷阱识别

当用户出现以下表述时，触发情绪警报：

**追涨表现：**
- "涨这么厉害，不买就错过了"
- " FOMO 发作了"
- "别人都在买"
- "这次不一样"

**恐慌表现：**
- "快跌死了，赶紧跑"
- "受不了了"
- "再不卖就归零了"
- "市场要崩盘了"

**报复交易：**
- "上把亏了，这把赚回来"
- "我就不信了"
- "必须干回来"

**过度自信：**
- "我这次看准了"
- "稳赚"
- "梭哈"
- "满仓干"

## 长期监督功能

### 交易记录追踪

在 memory 中记录：
- 每次审查结果
- 用户是否听从建议
- 后续实际盈亏
- 违规频率统计

### 定期复盘

每月提醒用户：
- 本月违规次数
- 因纪律避免的可能亏损
- 因冲动导致的事实亏损
- 纪律遵守率报告

## 与其他技能配合

- **stock-advisor**：提供基本面/估值分析数据
- **memory**：记录交易历史和审查记录
- **cron**：定时提醒复盘
